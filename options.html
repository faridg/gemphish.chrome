Welcome to Gemphisher! 🔍

Getting Started

  1. Create a Google Cloud account if you don't have one: Google Cloud Console
  2. Enable the Gemini API for your project
  3. Create an API key with access to Gemini API
  4. Copy your API key and paste it below

Your API key is stored locally and is only used to communicate with the Gemini API.

How to Use

  • Click the extension icon or right-click anywhere to open the analysis panel
  • The extension will analyze the current webpage for potential phishing risks
  • View the risk assessment and detailed analysis in the side panel
